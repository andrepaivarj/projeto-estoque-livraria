## Projeto Estoque de Livraria
Foi criado um projeto de gerenciamento de um estoque virtual de uma livraria, onde é possível criar (por meio de POO) um objeto Livro. Dentro desse objeto chamado Livro, o usuário fornece dados como:
 - Nome do livro;
 - Nome do Autor;
 - Ano de publicação;
 - Estoque do livro.

A partir desses dados, é possível:

- Alterar a quantidade de estoque;
- Adicionar novos livros ao catálogo.
